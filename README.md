# superstore-sales-analysis
Sales Data Analysis using Excel, MySQL and Power BI
## Tools Used
- Microsoft Excel (Data Cleaning & Pivot Tables)
- MySQL Workbench (SQL Queries)
- Power BI (Interactive Dashboard)

## Project Structure
- cleaned_sales_data.xlsx — Cleaned dataset
- sales_queries.sql — SQL queries
- dashboard_screenshot.png — Power BI Dashboard
- Business_Insights_Report.pdf — Business insights

## Key Findings
- West region has highest sales ($725,457)
- Technology is most profitable category
- Discounts above 20% cause losses
- November-December are peak months
